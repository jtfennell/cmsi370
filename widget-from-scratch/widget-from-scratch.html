Custom jQuery Slider By Jeff Fennell

Default

This custom jQuery widget allows the user to change the value of an input box via a slider. By default, the slider sets the input box's min and max values, to 0 and 100 respectively.

$('#money-input').slider();

Optional attributes

Color

The developer can also pass in an additional option to change the outline color of the slider knob.

$('#money-input').slider({ color: '#00FF00' });

Min and Max Values

The developer can also change the min and max values for the input box.

$('#money-input').slider({ color: 'red', min:100, max:200 });
